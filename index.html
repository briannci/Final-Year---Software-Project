PetPal

This is my 4th year Software Project. It is a hybrid application built using the Meteor.js framework. This application was built for the purpose of increasing the chances of finding your pet if reported missing. This application uses Meteor.js to inherit a real time application architecture.

  • User can register an account at /register and login at /login
  • The user can input details in relation to their pet and themselves
  • The above is then pushed to a public profile which is created for other users to view
  • /profile/username is where the profiles are then located
  • The founder of the pet can send the location to the pet owner

All of these features are completed in real time which is done based off the Meteor.js framework.
